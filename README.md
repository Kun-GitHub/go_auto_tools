### 2. 关于项目
本项目名称叫go-auto-tools，是一个关于Go语言开发的项目，但这个项目的开发语言却是Python，这有点意思，是吧。  

我先简单介绍这个项目的目标：
1. 帮助开发者快速构建一个Go语言项目，也就是Go语言的脚手架。
2. 能够自动化生成Go后端代码，目前考虑是基于（gorm+gin）
3. 会引入字节扣子，结合LLM大模型优化整个项目。

PS：这个项目开发进度会比较慢
